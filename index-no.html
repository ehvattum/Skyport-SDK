Her kan du laste ned siste versjonen av den offisielle Skyport utviklingspakken. Husk å verifisere signaturen etter nedlastingen, for å være sikker på at filen du lasted ned er ikke korrupt og fri av virus.
Sjekk at du har den siste versjonen av utviklingspakken fra denne siden, siden nyere versjoner kan bli lastet opp. Forumtråden AI Programming på forum.gathering.org vil også bli oppdatert hver gang det legges ut en ny versjon.

Komme i gang

All dokumentasjon er tilgjengelig på både norsk og engelsk. Start med å lese gjennom filen docs/en/GAME.md eller docs/no/GAME.md for den engelske eller norske versjonen av spillreglene. Ufullstendige APIer for protokollen i flere språk kan finnes i api/ mappen. For å komme i gang med å skrive din egen API (eller lese protokollen), les docs/en/PROTOCOL.md eller docs/no/PROTOCOL.md

    • Problemer med utdaterte java versjoner på OSX

    Det er et kjent problem med å kjøre serveren på OSX, som forårsakes av at OSX kan ha flere versjoner av java installert samtidig. I Safari får man den nyeste java 1.7 versjonen, men om man skriver `java` på kommando-linjen, startes java-versjon 1.6. Denne versjonen klarer ikke å kjøre serveren. En fiks for dette vil bli publisert her snart.
