Her kan du laste ned siste versjonen av den offisielle Skyport utviklingspakken. Husk å verifisere signaturen etter nedlastingen, for å være sikker på at filen du lasted ned ikke er korrupt og helt uten virus.
Husk også å sjekke at du alltid har den siste versjonen av utviklingspakken fra denne siden. Nyere versjoner kan komme med oppdateringer som vil påvirke din AI. Forumtråden AI Programming på forum.gathering.org vil også bli oppdatert hver gang det legges ut en ny versjon.

Komme i gang

All dokumentasjon er tilgjengelig på både norsk og engelsk. Start med å lese gjennom spill-reglene i docs/en/GAME.md (engelsk) eller docs/no/GAME.md (norsk). Ufullstendige APIer for protokollen i flere språk finner du også i api/ mappen. For å komme i gang med å skrive din egen API (eller lese protokollen), les docs/en/PROTOCOL.md (engelsk) eller docs/no/PROTOCOL.md (norsk)

    • Problemer med utdaterte java versjoner på OSX

    Serveren trenger en nyere java versjon en det som OSX tilbyr som standard. Det vil derfor være nødvendig å oppdatere dette manuelt. Merk at OSX også skiller på java som plugin (som brukes i nettlesere; firefox, safari osv) og java som et terminal-verktøy. Skyport serveren benytter seg av den sistnevnte. For å kunne ta i bruk serveren må man altså oppdatere terminal-java for å kunne gjennomføre kompileringen. En guide til dette vil bli publisert. Link til java.com for å sjekke plugin versjon Hvis du vil sjekke terminal versjonen: Åpne terminalen og skriv "java -version". Serveren trenger java versjon 1.7, mens nyeste OSX oppdatering tilbyr java versjon 1.6
