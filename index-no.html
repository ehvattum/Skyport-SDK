Her kan du laste ned siste versjonen av den offisielle Skyport SDKen. Vær sikkert å verifisere minst en av signaturene gitt nede etter nedlastingen, for å verifisere at filen du lasted ned er uskadet og virus-fri.
Sjekk at du har den siste versjonen av SDK-en fra denne siden, siden vi bestadig laster opp nye versjoner. Forum-tråden på gathering.org vil også bli oppdatert hver gang det legges ut en ny versjon.

Komme i gang

All dokumentasjon er tilgjengelig på både norsk og engelsk. Begynn med å lese filen docs/en/GAME.md eller docs/no/GAME.md for den engelske eller norske versjonen av spillreglene.
Uferdige APIer for protokollen for div. språk kan finnes i api/ mappen.
For å komme i gang med å skrive din egen API (eller lære om hvordan protokollet fungerer), les docs/en/PROTOCOL.md eller docs/no/PROTOCOL.md

    • Problemer med utdaterte java versjoner på OSX

    Det eksisterer et kjent problem med å kjøre servern på OSX, som forårsakes av at OSX har flere forskjellige java versjoner installert samtidig. I Safari/et al får man den nyeste java 1.7 versjonen, men om man skriver `java` på kommando-linjen, startes det en veldig gammel java 1.6 versjon, som ikke klarer å kjøre servern våres. En fiks for dette vil bli publisert her snart.
