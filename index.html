Here you can download the latest official Skyport development kit. Make sure you verify the signature after downloading, to ensure your download is not corrupted and virus-free.
Make sure you have the latest version of the SDK from this page, as newer versions are continuously released. The forum thread on forum.gathering.org will also be updated whenever a new release becomes available.

Getting started

All documentation is available in both english and norwegian. It is recommended that you start by reading the file docs/en/GAME.md or docs/no/GAME.md for the english and norwegian versions respectively, which describes the game rules. Incomplete APIs for the protocol for various languages can be found in the api/ folder. If you want to get started writing your own API (or in general looking into the protocol), read docs/en/PROTOCOL.md or docs/no/PROTOCOL.md for the english and norwegian versions respectively.

    Problems with outdated java versions on OSX

    There is a known problem running the server on OSX, which revolves around OSX having several different versions of java installed concurrently. In safari you get the latest java 1.7, but on the command-line, `java` invokes a very old java 1.6 version, which does not have the necessary feature-set required to run the server. A workaround for this issue will be published here soonish.
